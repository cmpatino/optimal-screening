# Optimal Screening Paper Code

This directory contains the camera-ready code needed to reproduce the paper figures and compute an optimal screening curve for a tabular dataset.

The research repository contains older exploratory code. This subproject is intentionally smaller: it keeps the simulation code, tabular baseline models, figure generation, and the core `compute_optimal_screening_curve` implementation.

## Setup

Install dependencies with uv:

```bash
uv sync
```

The project targets Python 3.11.

## Reproduce Paper Plots

Run the full no-input replication pipeline:

```bash
uv run replicate-plots
```

This writes JSON artifacts and PNG figures to `runs/`.

The command generates:

- `runs/simulations.json`
- `runs/utility_gap_vs_dist_param.json`
- `runs/screening_tabular_landmines_granada_test.json`
- `runs/screening_tabular_acs2018_all_strata_50pct.json`
- all figures generated by `scripts/plot_figures.py`

The ACS artifact uses `data/acs_income_2018.csv`. The smaller `data/acs_income_sample.csv` is included only for the `calculate-risk` example config.

## Manual Artifact Commands

The one-command pipeline runs these steps:

```bash
uv run python scripts/simulate_distributions.py
uv run python scripts/simulate_utility_gap.py
uv run python scripts/run_tabular_models.py --csv data/landmines_raw.csv --outcome mines_outcome \
  --strata Municipio --beta 0.1 --test-municipio GRANADA \
  --output runs/screening_tabular_landmines_granada_test.json
uv run python scripts/run_tabular_models.py --csv data/acs_income_2018.csv --outcome "PINCP > 50k" \
  --strata AGEP COW SCHL MAR OCCP POBP RELP WKHP SEX RAC1P --beta 0.3 --strata-only \
  --test-size 0.5 --output runs/screening_tabular_acs2018_all_strata_50pct.json
uv run python scripts/plot_figures.py --runs-dir runs/
```

## Compute An Optimal Screening Curve

Use a YAML or JSON config:

```bash
uv run calculate-risk configs/example-risk.yaml
```

The example config writes `runs/example-risk-output.json`.

Config fields:

- `csv`: input CSV path.
- `outcome`: binary outcome column.
- `strata`: columns used to estimate empirical risk strata.
- `beta`: treatment budget, between 0 and 1.
- `alpha_quantiles`: optional screening budgets; every value must be between 0 and `beta`.
- `risk_col`: optional column containing precomputed risk scores.
- `prediction_col`: optional model prediction column, default `probability`.
- `output`: optional output JSON path, default `runs/optimal_screening_curve.json`.

## Repository Map

- `src/optimal_screening/analysis/stratified.py`: core screening algorithms.
- `src/optimal_screening/cli/`: `replicate-plots` and `calculate-risk` entry points.
- `scripts/`: standalone artifact and plotting scripts.
- `configs/`: example risk-curve configuration.
- `data/`: minimal datasets needed by the included commands.
- `tests/`: focused unit tests for the core algorithm and risk CLI.
